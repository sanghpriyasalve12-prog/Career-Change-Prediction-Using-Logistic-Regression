# Career-Change-Prediction-Using-Logistic-Regression
This project analyzes career-related factors influencing an individual's likelihood of changing occupation. Using data cleaning, visualization, and logistic regression modeling, the study identifies key drivers behind career change decisions.
# Overview
This project demonstrates a complete data analytics workflow — from raw data to business insights. It covers data loading, exploratory data analysis (EDA), data cleaning and Final reporting.
The goal of the project is to showcase practical data skills used in real-world analytics roles.
# Dataset
The dataset contains structured career-related data used for analysis and insight generation.
- Files included:

Career data_PDA_4053.xlsx – Original raw dataset

career_data_cleaned.csv – Cleaned dataset used for analysis

career_data_pda.ipynb – Python notebook containing EDA and data processing

# Project Workflow / Steps
1.Data Loading

Imported dataset from Excel into Python environment.

2.Exploratory Data Analysis (EDA)

Checked data types, null values, and distributions

Identified trends, patterns, and outliers

3.Data Cleaning
Handled missing values
Removed duplicates
Standardized formats
Exported cleaned data as CSV
